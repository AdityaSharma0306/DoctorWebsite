Prescripto is an innovative doctor appointment booking application developed using MERN Stack (MongoDB, Express, React, Node.js) that aims to provide a seamless, intuitive, and efficient platform where patients can easily book, manage, cancel appointments and pay doctor’s fee with ease. 
The application also allows the admin to manage all appointments, add doctor, cancel appointment and appointment completion with proper authorization. The application provides a robust and scalable platform to manage the process of appointment booking.

Key Features :-
1. Appointment Booking: Patients can schedule appointments with available doctors.
2. Appointment Cancelation: Patients can cancel appointments if needed.
3. Fee Payment: Patients can pay the consultation fee for the doctor securely.
4. Appointment Completion: After completing  the  appointment  with  doctor  admin  can  complete  the  process of appointment booking.

This application aims to improve accessibility, reduce wait times, and streamline the management of appointments while also providing secure payment processing.
